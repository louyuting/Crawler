2016-8-3
	httpcore   是一套http协议实现包.实现了一系列的底层传输的功能,这些底层功能，可以用来去建立自己的client和server
	httpclient 是基于httpcore的一套客户端,httpclient是在httpcore上进行开发的,于是先看httpcore。

	可能是每次从流中取出数据太大，时间不够.
